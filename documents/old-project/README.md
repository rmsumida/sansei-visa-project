# Sansei Visa Project

A structured project to obtain, activate, and maintain a Japanese Long-Term Resident Visa based on 3rd-generation (Sansei) descent — operated entirely from the United States.

## Tooling

| Tool | Role |
|------|------|
| **Claude Chat** (Project) | Strategic discussion. System prompt lives in `PROJECT_INSTRUCTIONS.md`. |
| **Claude Code** | File management, session routines, project updates. Reads `CLAUDE.md` automatically. |
| **GitHub** | Single source of truth. Sync before every session, commit after. |
| **Obsidian** | Optional local reading/linking of project markdown files. |
| **VS Code** | Optional manual editing of project files. |
