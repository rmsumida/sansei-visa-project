# Sansei Visa Project

Strategic advisory project to obtain, activate, and legally maintain a Japanese Long-Term Resident Visa based on 3rd-generation (Sansei) descent — while remaining a full-time resident of Texas.

## Governance

This project uses a multi-session governance framework managed by Claude Code and Claude Chat. See `governance/` for project state, action items, decisions, and session logs.
